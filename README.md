# Data Structures & Algorithms Implementation

This repository contains my personal implementations of fundamental data structures, developed as part of the **CENG112 - Data Structures** course.

The implementations are based on the course curriculum and lecture slides, focusing on understanding the core logic behind memory management and algorithmic efficiency without relying on built-in libraries.
